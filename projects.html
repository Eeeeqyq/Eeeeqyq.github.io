ML for Student Debt Outcomes

GitHub

Group Project  ·  Northeastern University  ·  2026

End-to-end ML pipeline predicting institution-level years to student debt payoff using U.S. Department of Education College Scorecard data (6,300+ institutions). Engineered a custom target variable incorporating tax assumptions, BLS-calibrated living costs, loan amortization, and a 25-year IDR cap; prevented data leakage by excluding target inputs from features.

  • Trained and tuned four models: Ridge, Decision Tree, Random Forest, PyTorch Neural Network
  • Best performance: MAE ≈ 3.95 years  |  R² ≈ 0.63
  • Produced 20+ EDA and evaluation visualizations; deployed live interactive demo via Vercel
Python PyTorch scikit-learn pandas Matplotlib Vercel

Ambient Music Recommendation Agent

GitHub

Group Project  ·  Northeastern University  ·  2026

Goal-conditioned music recommendation system combining user context, wearable biosignals, and song representations across four mood states — focus, wind-down, uplift, movement.

  • Anchor-first retrieval and reranking pipeline with multitask prediction heads for intervention benefit and acceptance likelihood
  • Integrated multisource data (SiTunes, PMEmo, Spotify) with deterministic preprocessing and artifact-based validation for reproducible experiments
  • Held-out performance: Recall@20 = 0.746  |  Recall@50 = 0.881  |  Rerank Hit@10 = 0.717
